# Spring Boot Store

A simple backend REST API built with **Spring Boot** for managing store products.
This project was created for learning and practicing backend development concepts.

---

## Features
- CRUD operations for products
- RESTful API design
- Clean layered architecture (Controller, Service, Repository)
- Database integration using Spring Data JPA
- Exception handling
- Maven-based project structure

---

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Swagger
- Maven
- Git

---

## Setup & Run

1. Clone the repository
```bash
git clone https://github.com/mahmoud2617/Spring-boot-store.git
```

2. Open the project in IntelliJ IDEA

3. Configure database connection in application.yml

4. Run the application
```bash
mvn spring-boot:run
```
